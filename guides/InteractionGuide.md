## Interaction Guide!

Hello, this page is to help with interacting with the website. Please use the tips and tricks below to navigate.

### Toggle Pane
![TogglePane](guides/TogglePane.png)

Use the blank check box on the left to turn on a layer. If the layer boz has a green checkmark next to it, click the box again to turn the layer off.

### Time Slider
![TimeSlider](guides/TimeSlider.png)

Data has been collected from 2000-2005, and is binned by average over the year. Scroll left to right, or use the arrows to change years.

### Split View
![SplitView](guides/SplitView.png)

Using this functionallity opens a second pane with an independent toggle pane and time slider. Additionally, zoom features are linked so that layers can be compared side by side.

### Zoom 
If you zoom in, more information can be shown! Collection site circles will split up to be more granular and county boundaries become apparent 

Now, go have fun interacting!
